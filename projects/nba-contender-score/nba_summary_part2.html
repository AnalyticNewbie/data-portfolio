NBA ContenderScore — Model Validation & Refinement

This follow-up analysis evaluates the structural assumptions behind the ContenderScore metric, compares offensive and defensive signals, and tests whether non-linear models improve contender identification.

Offense vs. Defense — Which Matters More?

To isolate the predictive contribution of each side of the ball, two independent logistic regression models were trained using continuous relative efficiency metrics: Rel_ORtg and Rel_DRtg.

  • Outcome variable: reaching the Conference Finals or better
  • Data range: 2005–2018 NBA seasons
  • Evaluation: Pseudo R-squared and coefficient magnitude

Conclusion: Defensive efficiency is a slightly stronger single predictor of contender status than offensive efficiency.

Improving Recall — Structural & Non-Linear Fixes

The original ContenderScore model achieved high overall accuracy but suffered from low recall, missing a large proportion of true contenders. Two approaches were tested to address this.

Test 1: Unconstrained Logistic Regression

A logistic regression model was fitted using both offensive and defensive metrics simultaneously, removing the fixed weighting imposed by the original ContenderScore formula.

Pseudo R-squared

0.4277

Recall

0.4423

This model significantly improved statistical fit, confirming that the original linear weighting constrained predictive performance.

Test 2: Balanced Random Forest

To capture non-linear interactions and address class imbalance, a Random Forest model was trained with class weighting to prioritise recall.

Final Recall (Balanced Random Forest)

62.5%

Conclusion: Non-linear models materially improve contender identification by reducing false negatives, at the cost of some precision.
