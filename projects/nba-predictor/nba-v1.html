The MVP

Every project needs a starting point. For v1.0, the goal was simple: Ingest data from the NBA API and output a reasonable prediction.

Methodology

I used a statistical approach based on Net Ratings (Points Scored - Points Allowed per 100 possessions).

  • Calculated Z-Scores for every team's offense and defense.
  • Added a flat constant for Home Court Advantage.
  • Simple linear projection for the final score.

Tech Stack

  • Python (Pandas)
  • SQLite (Lightweight DB)
  • NBA API
  • Cron (Automation)
