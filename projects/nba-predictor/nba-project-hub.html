NEW UPDATE

v2.2: Physics & Psychology

December 18, 2025

AUC: 0.725

A major leap forward. We identified the "Monday Anomaly" (Home Court Advantage disappears on Mondays) and integrated Physics engines for Pace and Rebounding. This allows the model to differentiate between "Track Meets" and "Grindfests," improving Total Score accuracy by ~15%.

v2.1: The Sunday Fix

December 14, 2025

AUC: 0.680

Corrected a data ingestion error where "Sunday Travel" games were mislabeled. This update stabilized the volatility metrics and introduced the first version of the "Rest Disadvantage" feature.

v2.0: The Migration

December 01, 2025

AUC: 0.645

Moved from simple linear regression to Gradient Boosting Classifiers. Established the PostgreSQL database schema and automated the daily data ingestion pipeline from the NBA API.

v1.0: MVP Launch

November 2025

Baseline

Initial proof of concept using basic rolling averages to predict game winners. Proved that automated data collection was viable.
